How to Use the Rich Text Generator: A Simple Guide

  1. Add Your Text and Colors: In the Text fields, type the words or phrases you want to use. Use the corresponding Color buttons to select a color for each text segment. Click Add Text Row to add more.
  2. Style Individual Text Segments: For each text row, you can apply Bold, Italic, Underline, or Strikethrough by checking the boxes right below the color picker.
  3. Choose Global Formatting: Use the options below to apply formatting to the entire text.
    • Rainbow: Select this to apply a rainbow effect. This will override all individual colors.
    • Font: Choose a Font Style and Font Weight from the dropdowns.
    • Size: Enter a number to change the font size.
    • Stroke: Check "Enable Stroke" and choose a color and thickness for an outline.
    • Transparency: Enter a number between 0 and 1 for text transparency.
  4. Generate and Copy: As you adjust settings, the Rich Text Code and the Preview will update automatically. When you are ready, click Copy to Clipboard.
  5. Toggle Dark Mode: Click the Light Mode or Dark Mode button in the top right corner to switch the website theme.

The code you copy can be used directly in any Roblox UI element that supports rich text, like a TextLabel or a TextBox.
